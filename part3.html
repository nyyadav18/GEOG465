For my points, I chose the options autoPan and keyboard. I picked autoPan because it helps focus the viewer on the marker selected by zooming to that location. I picked keyboard because it allows for other ways to access points on the map without the trackpad, or a mouse.

For the polylines, I felt like my choices for options not discussed in class were pretty limited, so I used both the smoothFactor option and the noClip option. I honestly tried several values in both options, for example I changed the smoothFactor from 1.0-10.0 and noticed no difference so I will say I am not entirely certain what they do. The documentation offers an explanation for the options though, smoothFactor lets the webmap know how to simplify the object based on the zoom level, while noClip, when set to true, makes sure the entire polyline is visible even if it goes beyond map view.

Finally, for my polygons, the additional option I chose was smoothFactor. The lower the smoothFactor the smoother the outline will be, the higher it is the less smooth the object will be.
